# 雾霾检测上位机
## 简介
* 用py3.6以及pyqt5写的上位机

## 支持的硬件及系统
* 上位机: Windows 10 x64
* 下位机: 树莓派3代B型 + Raspbian20171129 + RaspberryPiCamera官方摄像头

## 需求
* python 3.6
* PyQt5
* OpenCv
* numpy


## 运行方法
* 上位机: run.py
* 下位机: src/pi_client.py
